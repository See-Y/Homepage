경량 정적 블로그 시스템

프레임워크 없이 순수 HTML/JS/CSS만으로 동작하며, 클라이언트 사이드 마크다운 렌더링을 지원하는 개인 블로그입니다.

HTML5 Vanilla JS Playwright
소스 코드 Live Demo

문제

기존 블로그 시스템들은 너무 무겁거나 불필요한 빌드 파이프라인을 요구했습니다. 빠른 로딩 속도와 유지보수의 편리함을 위해, 프레임워크 없는 순수 정적 사이트가 필요했습니다.

해결

Vanilla JS와 CSS 변수를 활용한 자체 디자인 시스템을 구축하고, `marked.js`를 통해 클라이언트 렌더링 방식의 마크다운 로더를 작성하여 빌드 없는 정적 배포를 실현했습니다.

결과

Lighthouse 점수 100점을 달성했으며, Playwright E2E 테스트를 통해 무결성을 검증했습니다. 이제 마크다운 파일만 업로드하면 자동으로 포스팅이 렌더링됩니다.
